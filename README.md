Iris_Classifier

Description:
Iris Flower Classification using PyTorch. A neural network predicts the species (Setosa, Versicolor, Virginica) from sepal and petal measurements. Includes data preprocessing and model training for supervised learning.

Model

Input layer: 4 features

Hidden layers: 8 → 4 → 3

Activation: ReLU

Output: Softmax via CrossEntropyLoss

How to Run
pip install torch pandas scikit-learn
python iris_nn.py
